# Node.js & Express Interview Preparation

This section covers Node.js core concepts, asynchronous patterns, Express.js architecture, and security best practices.

## Topics Covered

1. **Core Concepts**: Event loop, modules (CommonJS vs ES), and buffers/streams.
2. **Asynchronous Patterns**: Callbacks, Promises, async/await, and EventEmitter.
3. **REST APIs with Express**: Routing, middleware, and request/response handling.
4. **Security & Middleware**: JWT, CORS, helmet, and error-handling middleware.

Each sub-topic contains:

- `OnlyQuestions.md`: A list of questions for self-assessment.
- `Questions_and_Answers.md`: Questions followed by detailed explanations and answers.
