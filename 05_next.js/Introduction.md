# Next.js Interview Preparation

This section covers Next.js specific features, routing (App Router), data fetching, rendering strategies, and optimizations.

## Topics Covered

1. **Routing**: File-based routing, App Router, segments, and navigation.
2. **Data Fetching**: Fetching on the server and client, caching, and revalidation.
3. **Rendering Strategies**: SSR, SSG, ISR, and CSR in the context of Next.js.
4. **Optimizations**: Image optimization, font optimization, and script management.

Each sub-topic contains:

- `OnlyQuestions.md`: A list of questions for self-assessment.
- `Questions_and_Answers.md`: Questions followed by detailed explanations and answers.
