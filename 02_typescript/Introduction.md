# TypeScript Interview Preparation

This section covers TypeScript concepts from basic syntax and types to advanced generics, decorators, and best practices.

## Topics Covered

1. **Basics**: Core concepts, compilation, and basic types.
2. **Type System**: Union/intersection types, type aliases, and literal types.
3. **Interfaces**: Defining and extending interfaces.
4. **Generics**: Type parameters, constraints, and reusable components.
5. **Advanced Types**: Conditional types, mapped types, and utility types.
6. **Type Guards & Narrowing**: Techniques for refining types at runtime.
7. **Classes**: OOP with TypeScript members and modifiers.
8. **Modules & Namespaces**: Organizing code in larger applications.
9. **Declaration Files**: Working with external JS libraries and ambient types.
10. **Enums**: Grouping related constants.
11. **Decorators**: Meta-programming using class and method decorators.
12. **Error Handling & Strict Mode**: Ensuring type safety and handling exceptions.
13. **Practical & Best Practices**: Real-world scenarios (React, API) and immutability.
14. **Coding Challenges**: Implementation exercises for TypeScript patterns.

Each sub-topic contains:

- `OnlyQuestions.md`: A list of questions for self-assessment.
- `Questions_and_Answers.md`: Questions followed by detailed explanations and answers.
