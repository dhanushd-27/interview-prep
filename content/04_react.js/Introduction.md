# React.js Interview Preparation

This section covers core React concepts, hooks, state management, and performance optimization techniques.

## Topics Covered

1. **Hooks**: In-depth look at `useState`, `useEffect`, `useMemo`, `useCallback`, and custom hooks.
2. **Components & Props**: Functional components, props, children, and composition.
3. **State Management**: Local state, Context API, and global state considerations.
4. **Performance**: Reconciliation, memoization, and optimizing renders.

Each sub-topic contains:

- `OnlyQuestions.md`: A list of questions for self-assessment.
- `Questions_and_Answers.md`: Questions followed by detailed explanations and answers.
